Migrating to Canopy
###################

If you've found that your existing application matches Canopy :ref:`use_cases_topic`, then you're ready to learn how to make the two work together.

* :ref:`Transitioning to Canopy <transitioning_mt>` has tons of information, from updating schemas and queries to step-by-step instructions for Rails and Django apps.
* To help get the most performance out of a new application on Canopy, check out the tips in :ref:`performance_tuning`, including more about table distribution columns.
* Finally, read :ref:`production_sizing` to help decide the best cluster size for a new application.
