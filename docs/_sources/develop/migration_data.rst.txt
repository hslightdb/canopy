Migrate Production Data
=======================

At this time, having updated the database schema and application queries to work with Canopy, you're ready for the final step. It's time to migrate data to the Canopy cluster and cut over the application to its new database.

The data migration path is dependent on downtime requirements and data size, but generally falls into one of the following two categories.
